# 🚀 假如我是马斯克，我能买下什么？

一个有趣的**虚拟富翁购物体验**网站。你可以扮演马斯克、马云、马化腾、黄仁勋等 8 位亿万富翁，在购物车里尽情买买买，体验一把"花钱如流水"的感觉！

## 功能特色

- 🎭 **8 位名人角色**：马斯克、马云、马化腾、黄仁勋、盖茨、巴菲特、张一鸣、钟睒睒
- 🛍️ **9 大品类 60+ 件商品**：从星巴克到 Twitter，从小商品到大公司
- 💰 **实时财富递减动画**：看着钱一点点变少（或者变负 😅）
- 🏦 **欠揍银行贷款系统**：花超了？王得花行长欢迎您！（年利率 666%）
- 🖼️ **纪念账单图片生成**：一键生成购物纪念账单，保存分享
- 🎉 **Confetti 庆祝动画**：结账时漫天彩纸
- 😂 **海量彩蛋文案**：每个商品都有专属幽默评价

## 使用方式

直接打开 `index.html` 即可体验，无需任何服务器环境。
